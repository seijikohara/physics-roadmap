<h1 align="center">Physics Roadmap</h1>

<p align="center">前提知識から積み上げ、理論物理学の到達目標を数式として読み解く、自己完結した学習ロードマップ。</p>

公開サイト: <https://seijikohara.github.io/physics-roadmap>

## 概要

物理学を分野ごとのカテゴリに分け、前提知識を起点に依存順へ積み上げる学習ロードマップである。各分野を、外部資料に頼らず一連の章だけで到達目標まで導く。到達目標は理論物理学の数式理解に置く。

## 特徴

- **自己完結**: 説明の論理を外部資料に委ねない。前提に含まれない数学も、使う前に章内で導入する。
- **演習による習得**: 各章に例題と演習問題を置く。演習には解答を付ける。
- **一次資料への脚注**: さらに学ぶための一次資料・データ出典を、章末の参考文献に示す。

全体設計は[docs/roadmap-design.md](docs/roadmap-design.md)に定める。
