## 前提

本章の前提は、次の 2 つである。

- [複素数の定義と四則演算](../definition-and-arithmetic/)：虚数単位 $i$、複素数 $a + bi$、共役 $\overline{z}$、積 $z\overline{z} = a^2 + b^2$ を使う。
- [平面ベクトルの定義と成分表示](../../vectors/plane-vectors-and-components/)：成分 $(a, b)$ と大きさ $\sqrt{a^2 + b^2}$ の対応に使う。

加えて、[前提とする知識](../../../prerequisites/knowledge/)の座標平面・点の座標 $(x, y)$・三平方の定理を用いる。

複素数カテゴリの第 2 章である。第 1 章では複素数を代数的な対象として扱った。本章では複素数を平面上の点として図示し、図形と結びつけて理解を深める。

## 学習目標

本章を読むと、次の概念と記号を使えるようになる。

- **複素平面**（ガウス平面）の **実軸**・**虚軸**と、複素数 $z = a + bi$ を点 $(a, b)$ として図示する方法
- 実部 $\operatorname{Re}(z)$・虚部 $\operatorname{Im}(z)$ の図上での読み取り
- 複素数と平面ベクトルの対応、および和・差の図形的意味（矢印・平行四辺形）
- **絶対値** $\lvert z \rvert = \sqrt{a^2 + b^2}$ と、原点からの距離としての意味
- 2 点間の距離 $\lvert z_1 - z_2 \rvert$
- **共役** $\overline{z}$ の実軸に関する対称性、関係式 $\lvert z \rvert^2 = z\overline{z}$・$\lvert \overline{z} \rvert = \lvert z \rvert$

## 複素平面

### 平面への対応づけ

複素数 $z = a + bi$ は、実部 $a$ と虚部 $b$ の 2 つの実数で定まる。前章で定めたとおり、実部を $\operatorname{Re}(z) = a$、虚部を $\operatorname{Im}(z) = b$ で表す。$\operatorname{Re}(z)$ は「$z$ の実部」、$\operatorname{Im}(z)$ は「$z$ の虚部」と読む。いずれも実数であり、虚部 $\operatorname{Im}(z) = b$ は「$bi$」ではなく実数 $b$ を指す。

順序づけられた実数の組 $(a, b)$ は、座標平面上の 1 点を表す。よって複素数 $z = a + bi$ に、座標平面上の点 $(a, b)$ をただ 1 つ対応させられる。

逆に、座標平面上の点 $(a, b)$ には複素数 $a + bi$ がただ 1 つ対応する。複素数と平面上の点は、相等の規則を通じて 1 対 1 に対応する。

> **複素平面の定義**：複素数 $z = a + bi$ を点 $(a, b)$ として図示する座標平面を、**複素平面** または **ガウス平面** と呼ぶ[^gauss]。横軸を **実軸**、縦軸を **虚軸** と呼ぶ。

軸の意味を述べる。

- **実軸**（横軸）：点の横座標が実部 $a$ を表す。実軸上の点 $(a, 0)$ は実数 $a = a + 0i$ に対応する。
- **虚軸**（縦軸）：点の縦座標が虚部 $b$ を表す。実部が $0$ で、虚部は $0$ でない複素数を **純虚数** と呼ぶ。虚軸上の原点以外の点 $(0, b)$（$b \ne 0$）は純虚数 $bi = 0 + bi$ に対応する。

実軸の名前は $\operatorname{Re}$、虚軸の名前は $\operatorname{Im}$ で表す。原点 $(0, 0)$ は複素数 $0 = 0 + 0i$ に対応する。

## 複素数と平面ベクトル

### 矢印としての複素数

複素数 $z = a + bi$ には点 $(a, b)$ が対応する。点 $(a, b)$ には、前提の章で学んだ成分 $(a, b)$ の平面ベクトルも対応する。すなわち原点 $\mathrm{O}$ を始点とし、点 $(a, b)$ を終点とする矢印が対応する。

> **複素数とベクトルの対応**：複素数 $z = a + bi$ に、成分 $(a, b)$ の平面ベクトルを対応させる。原点から点 $(a, b)$ へ向かう矢印が複素数 $z$ を表す。

複素数の和に対応する点を求める。第 1 章の和の公式から、和の実部・虚部はそれぞれ実部どうし・虚部どうしの和である。

$$
z_1 + z_2 = (a_1 + a_2) + (b_1 + b_2)i
$$

よって和 $z_1 + z_2$ には、点 $(a_1 + a_2,\ b_1 + b_2)$ が対応する。

### 和の図形的意味

複素数の和 $z_1 + z_2$ は、**平行四辺形** を使って図示できる。次の 4 点を考える。

- 原点 $\mathrm{O} = (0, 0)$
- 点 $z_1 = (a_1, b_1)$
- 点 $z_2 = (a_2, b_2)$
- 点 $z_1 + z_2 = (a_1 + a_2,\ b_1 + b_2)$

4 点が平行四辺形を作ることを、座標から確かめる。

- 点 $z_1$ から点 $z_1 + z_2$ への進みは、横が $(a_1 + a_2) - a_1 = a_2$、縦が $(b_1 + b_2) - b_1 = b_2$ である。原点 $\mathrm{O}$ から点 $z_2$ への進み $(a_2, b_2)$ と一致する。
- よって辺 $z_1$–$(z_1 + z_2)$ は、辺 $\mathrm{O}$–$z_2$ と向き・長さが等しい。前提の章のベクトルの相等により、向きが等しい 2 辺は平行で、長さも等しい。
- 同じ計算で、辺 $z_2$–$(z_1 + z_2)$ は辺 $\mathrm{O}$–$z_1$ と平行で長さが等しい。

向かい合う 2 組の辺は、それぞれ平行で長さが等しい。よって 4 点 $\mathrm{O}$・$z_1$・$(z_1 + z_2)$・$z_2$ は平行四辺形を作る。原点から対角の頂点 $z_1 + z_2$ へ向かう対角線が、和を表す。

平行四辺形の図から、和には平行移動の意味も読み取れる。点 $z_1$ に複素数 $z_2$ を足すと、点 $z_1$ が矢印 $z_2$ の分だけ平行移動して点 $z_1 + z_2$ に移る。すなわち、ある点に複素数 $w$ を足す操作は、点を矢印 $w$ の分だけ平行移動する操作にあたる。

例として $z_1 = 3 + i$ と $z_2 = 1 + 2i$ の和を図示する。和は $z_1 + z_2 = (3 + 1) + (1 + 2)i = 4 + 3i$ である。

### 差の図形的意味

差 $z_1 - z_2$ も実部どうし・虚部どうしの差で定まる。

$$
z_1 - z_2 = (a_1 - a_2) + (b_1 - b_2)i
$$

差 $z_1 - z_2$ には、点 $z_2$ を始点とし点 $z_1$ を終点とする矢印が対応する。理由を述べる。差の定め方から、$z_2$ に $z_1 - z_2$ を足すと $z_1$ になる。

$$
z_2 + (z_1 - z_2) = z_1
$$

足し算は矢印の分の平行移動であった。点 $z_2$ を矢印 $z_1 - z_2$ の分だけ平行移動すると、点 $z_1$ に移る。よって矢印 $z_1 - z_2$ は、点 $z_2$ から点 $z_1$ へ向かう向きと長さを持つ。

例として $z_1 = 2 + 3i$ と $z_2 = 4 + i$ の差を図示する。差は $z_1 - z_2 = (2 - 4) + (3 - 1)i = -2 + 2i$ である。

## 絶対値

### 原点からの距離

複素数 $z = a + bi$ の絶対値 $\lvert z \rvert = \sqrt{a^2 + b^2}$ を、本章で定める。複素平面の上では、絶対値は明確な図形的意味を持つ。

> **絶対値の図形的意味**：$\lvert z \rvert$ は、原点 $\mathrm{O}$ から点 $z$ までの距離である。

距離が公式と一致することを、三平方の定理から確かめる。点 $z = a + bi$ は座標 $(a, b)$ にある。原点 $(0, 0)$ から点 $(a, b)$ までの線分を斜辺とする直角三角形を作る。

- 横の辺：原点 $(0, 0)$ から点 $(a, 0)$ までで、長さは $\lvert a \rvert$ である。
- 縦の辺：点 $(a, 0)$ から点 $(a, b)$ までで、長さは $\lvert b \rvert$ である。
- 斜辺：原点 $(0, 0)$ から点 $(a, b)$ までで、長さが求める距離である。

三平方の定理により、斜辺の長さの 2 乗は $\lvert a \rvert^2 + \lvert b \rvert^2 = a^2 + b^2$ である。距離は $0$ 以上なので、正の平方根を取って $\sqrt{a^2 + b^2}$ を得る。代数的な定義 $\lvert z \rvert = \sqrt{a^2 + b^2}$ と一致する。

例として $z = 3 + 4i$ の絶対値を図示する。$\lvert z \rvert = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$ である。

絶対値の基本的な性質を 2 つ挙げる。いずれも図形的意味から読み取れる。

- $\lvert z \rvert \ge 0$ である。距離は負にならない。
- $\lvert z \rvert = 0$ となるのは $z = 0$ のときに限る。距離が $0$ となるのは、点が原点に重なるときだけである。

### 2 点間の距離

差の図形的意味と絶対値を組み合わせると、2 点間の距離が得られる。複素数 $z_1 = a_1 + b_1 i$ と $z_2 = a_2 + b_2 i$ を考える。差は次のとおりである。

$$
z_1 - z_2 = (a_1 - a_2) + (b_1 - b_2)i
$$

差の絶対値に公式を当てる。

$$
\lvert z_1 - z_2 \rvert = \sqrt{(a_1 - a_2)^2 + (b_1 - b_2)^2}
$$

右辺の値は、横の差 $a_1 - a_2$ と縦の差 $b_1 - b_2$ を直角を挟む 2 辺とする直角三角形の斜辺の長さである。三平方の定理により、斜辺の長さは 2 点 $(a_1, b_1)$・$(a_2, b_2)$ の距離に等しい。よって次が成り立つ。

> **2 点間の距離**：$\lvert z_1 - z_2 \rvert$ は、複素平面上の点 $z_1$ と点 $z_2$ の距離に等しい。

差を表す矢印は、点 $z_2$ から点 $z_1$ へ向かう。矢印の長さが、そのまま 2 点間の距離 $\lvert z_1 - z_2 \rvert$ である。

## 共役

### 実軸に関する対称

前章で定めたとおり、複素数 $z = a + bi$ の共役は虚部の符号を変えた $\overline{z} = a - bi$ である（$\overline{z}$ は「$z$ バー」と読む）。実部は変わらず、虚部の符号だけが反転する。座標で見ると、点 $z = (a, b)$ と点 $\overline{z} = (a, -b)$ は、横座標が等しく縦座標の符号だけが逆である。

> **共役の図形的意味**：点 $\overline{z}$ は、点 $z$ を **実軸に関して対称に折り返した点** である。

例として $z = 3 + 2i$ と共役 $\overline{z} = 3 - 2i$ を図示する。2 点は実軸（横軸）を鏡として向かい合う。

### 絶対値と共役の関係

共役と絶対値の間には、簡潔な関係が成り立つ。

**関係 1：$\lvert \overline{z} \rvert = \lvert z \rvert$。** 共役 $\overline{z} = a - bi$ の絶対値を計算する。

$$
\lvert \overline{z} \rvert = \sqrt{a^2 + (-b)^2} = \sqrt{a^2 + b^2} = \lvert z \rvert
$$

虚部を 2 乗すると符号が消えるため、絶対値は変わらない。図でも、$z$ と $\overline{z}$ は原点から等距離にある。

**関係 2：$\lvert z \rvert^2 = z\overline{z}$。** 積 $z\overline{z}$ を、$i^2 = -1$ を用いて展開する。

$$
z\overline{z} = (a + bi)(a - bi) = a^2 - (bi)^2 = a^2 - b^2 i^2 = a^2 + b^2
$$

右辺 $a^2 + b^2$ は $\lvert z \rvert^2$ である。よって次が成り立つ。

$$
\lvert z \rvert^2 = z\overline{z}
$$

関係 2 は重要である。複素数の絶対値の 2 乗を、平方根を使わずに積 $z\overline{z}$ で表せる。積 $z\overline{z}$ は実数 $a^2 + b^2$ になり、虚部を持たない。

実部・虚部も共役を使って表せる。共役の定義から $z + \overline{z} = 2a$、$z - \overline{z} = 2bi$ である。よって次が成り立つ。

$$
\operatorname{Re}(z) = \frac{z + \overline{z}}{2}, \qquad \operatorname{Im}(z) = \frac{z - \overline{z}}{2i}
$$

## 例題

### 例題 1

複素数 $z = -2 + 3i$ を複素平面上に図示する位置を述べよ。また実部と虚部を答えよ。

**解法.** $z = -2 + 3i$ の実部は $\operatorname{Re}(z) = -2$、虚部は $\operatorname{Im}(z) = 3$ である。よって点 $(-2, 3)$ に対応する。実軸方向へ $-2$（左へ $2$）、虚軸方向へ $3$（上へ $3$）進んだ位置に点を打つ。

### 例題 2

複素数 $z = -5 + 12i$ の絶対値 $\lvert z \rvert$ を求めよ。

**解法.** 絶対値の公式に $a = -5$・$b = 12$ を代入する。2 乗すると符号が消える。

$$
\lvert z \rvert = \sqrt{(-5)^2 + 12^2} = \sqrt{25 + 144} = \sqrt{169} = 13
$$

### 例題 3

2 つの複素数 $z_1 = 5 + 6i$ と $z_2 = 1 + 3i$ の距離 $\lvert z_1 - z_2 \rvert$ を求めよ。

**解法.** まず差を計算する。

$$
z_1 - z_2 = (5 - 1) + (6 - 3)i = 4 + 3i
$$

差の絶対値が 2 点間の距離である。

$$
\lvert z_1 - z_2 \rvert = \sqrt{4^2 + 3^2} = \sqrt{16 + 9} = \sqrt{25} = 5
$$

### 例題 4

任意の複素数 $z_1$・$z_2$ について、$\lvert z_1 z_2 \rvert = \lvert z_1 \rvert \lvert z_2 \rvert$ が成り立つことを示せ。関係式 $\lvert z \rvert^2 = z\overline{z}$ を使ってよい。

**解法.** まず、積の共役が共役の積に等しいこと、すなわち $\overline{z_1 z_2} = \overline{z_1}\,\overline{z_2}$ を確かめる。$z_1 = a + bi$・$z_2 = c + di$ とおく。積は次のとおりである。

$$
z_1 z_2 = (ac - bd) + (ad + bc)i
$$

積の共役は、虚部の符号を変えて $\overline{z_1 z_2} = (ac - bd) - (ad + bc)i$ である。一方、共役の積を展開する。

$$
\overline{z_1}\,\overline{z_2} = (a - bi)(c - di) = (ac - bd) - (ad + bc)i
$$

両者は一致する。よって $\overline{z_1 z_2} = \overline{z_1}\,\overline{z_2}$ である。次に、関係式 $\lvert z \rvert^2 = z\overline{z}$ を $z_1 z_2$ に当てる。

$$
\lvert z_1 z_2 \rvert^2 = (z_1 z_2)\overline{(z_1 z_2)} = z_1 z_2 \,\overline{z_1}\,\overline{z_2} = (z_1 \overline{z_1})(z_2 \overline{z_2}) = \lvert z_1 \rvert^2 \lvert z_2 \rvert^2
$$

両辺は $0$ 以上である。正の平方根を取ると $\lvert z_1 z_2 \rvert = \lvert z_1 \rvert \lvert z_2 \rvert$ を得る[^mult]。

<details>

$$

### 問題 3

2 つの複素数 $z_1 = -2 + 4i$ と $z_2 = 1 - i$ の距離 $\lvert z_1 - z_2 \rvert$ を求めよ。

<details>
<summary>解答を表示</summary>

まず差を計算する。

$$
z_1 - z_2 = (-2 - 1) + (4 - (-1))i = -3 + 5i
$$

差の絶対値が 2 点間の距離である。

$$
\lvert z_1 - z_2 \rvert = \sqrt{(-3)^2 + 5^2} = \sqrt{9 + 25} = \sqrt{34}
$$

$\sqrt{34}$ は平方数でないため、根号のまま残す。

</details>

### 問題 4

複素数 $z = a + bi$ について、積 $z\overline{z}$ が常に $0$ 以上の実数になることを示せ。

<details>
<summary>解答を表示</summary>

積 $z\overline{z}$ を、$i^2 = -1$ を用いて展開する。

$$
z\overline{z} = (a + bi)(a - bi) = a^2 - b^2 i^2 = a^2 + b^2
$$

$a$・$b$ は実数なので $a^2 \ge 0$・$b^2 \ge 0$ である。よって $z\overline{z} = a^2 + b^2 \ge 0$ であり、虚部を持たない実数になる。

</details>

### 問題 5

絶対値が $\lvert z \rvert = 2$ を満たす複素数 $z$ が表す点は、複素平面上でどのような図形を描くか答えよ。

<details>
<summary>解答を表示</summary>

$\lvert z \rvert$ は原点から点 $z$ までの距離である。$\lvert z \rvert = 2$ は、原点からの距離が常に $2$ であることを表す。

原点からの距離が一定 $2$ の点の集まりは、原点を中心とする **半径 $2$ の円** である。

</details>

## まとめ

本章は、複素数を複素平面の上で図示し、絶対値と共役の図形的意味を自己完結で導入した。要点を振り返る。

- **複素平面**：複素数 $z = a + bi$ を点 $(a, b)$ として図示する。横軸が実軸、縦軸が虚軸である。
- **ベクトルとの対応**：複素数 $z = a + bi$ は成分 $(a, b)$ の平面ベクトルに対応する。和は平行四辺形、差は 2 点を結ぶ矢印で表せる。
- **絶対値**：$\lvert z \rvert = \sqrt{a^2 + b^2}$ は原点から点 $z$ までの距離である。距離 $\lvert z_1 - z_2 \rvert$ は 2 点間の距離になる。
- **共役**：$\overline{z}$ は点 $z$ を実軸に関して折り返した点である。$\lvert \overline{z} \rvert = \lvert z \rvert$ と $\lvert z \rvert^2 = z\overline{z}$ が成り立つ。

次の章[複素数の極形式](../polar-form/)では、複素数を原点からの距離 $\lvert z \rvert$ と、実軸からの角度（偏角）の 2 つで表す。本章で定めた絶対値が、極形式の「距離」の部分にあたる。角度を導入すると、積と商の図形的意味が明快になる。

[^gauss]: 「ガウス平面」の名は、複素数の幾何的な表示を体系的に用いた数学者カール・フリードリヒ・ガウス（C. F. Gauss）にちなむ。同じ表示を独立に論じたカスパー・ウェッセル（C. Wessel）とジャン＝ロベール・アルガン（J.-R. Argand）の名から、「アルガン図」と呼ぶ流儀もある。

[^mult]: 等式 $\lvert z_1 z_2 \rvert = \lvert z_1 \rvert \lvert z_2 \rvert$ は、絶対値が積を保つことを示す。第 3 章の極形式で複素数を「距離」と「角度」に分けると、積の絶対値が距離どうしの積になる理由がさらに明快になる。
