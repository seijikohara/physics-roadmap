## 前提

本章の前提は、次の 3 つである。

- [判別式](../../algebra/discriminant/)：判別式 $D = b^2 - 4ac$ の定義と、$D$ の符号による二次方程式の実数解の個数を使う。
- [二次関数](../quadratic-functions/)：二次関数 $y = ax^2 + bx + c$・放物線・頂点・軸・標準形・下に凸と上に凸の区別を使う。
- [一次不等式](../../algebra/linear-inequalities/)：不等式の基本性質（負の数を掛けると向きが反転する規則）、絶対値を含む不等式 $\lvert x \rvert < a$・$\lvert x \rvert > a$、同値記号 $\iff$ を使う。

前章の[二次関数](../quadratic-functions/)では、放物線の形と頂点を扱った。本章は、放物線が $x$ 軸の上にあるか下にあるかを読み取り、二次不等式を解く。さらに、絶対値を含む不等式へ進む。

物理学では、量に範囲の条件を課す場面が多い。例として、放物運動の物体が一定の高さを超える時刻の範囲や、測定値が許容誤差に収まる条件を考える。範囲を表す条件は不等式で書く。本章は二次の不等式と絶対値の不等式を、前提の章だけから自己完結で解く。

## 学習目標

本章を読むと、次の概念と記号を使えるようになる。

- 二次不等式 $ax^2 + bx + c > 0$ などを、放物線と $x$ 軸の上下で解く方法
- 判別式 $D$ の符号（$D > 0$・$D = 0$・$D < 0$）による解の形の場合分け
- $x$ 軸との交点が 2 個・1 個（接する）・0 個の各場合の解
- 絶対値を含む不等式 $\lvert x \rvert < a$・$\lvert x \rvert > a$ の解
- 一般の関数を含む絶対値不等式 $\lvert f(x) \rvert < a$ の解法

## 二次不等式とは

### 定義

未知数について 2 次式である不等式を**二次不等式**と呼ぶ。次はいずれも未知数 $x$ の二次不等式である。

$$
x^2 - 3x + 2 > 0, \qquad -x^2 + 4 \le 0, \qquad 2x^2 + x - 1 \ge 0
$$

二次不等式を**解く**とは、不等式を成り立たせる $x$ の値をすべて求める操作である。解はふつう値の範囲になる。

本章では、まず不等式を $ax^2 + bx + c$ の符号の問題に書き直す。$a < 0$ のときは両辺に $-1$ を掛けて $a > 0$ にそろえられる。負の数を掛けると不等号の向きは反転する[^sign-flip]。よって以下では、特に断らない限り $a > 0$ として説明する。

### 符号と放物線の上下

二次不等式 $ax^2 + bx + c > 0$ は、二次関数 $y = ax^2 + bx + c$ の値 $y$ が正になる $x$ を問う不等式である。前提の章[二次関数](../quadratic-functions/)で、二次関数のグラフは放物線になることを学んだ。値 $y$ の符号は、放物線の上下で読み取れる。

> 二次関数 $y = ax^2 + bx + c$ のグラフを使うと、二次不等式の符号は次のように読める。
>
> - $y > 0$ となる $x$ は、放物線が $x$ 軸より**上**にある部分の $x$ である。
> - $y < 0$ となる $x$ は、放物線が $x$ 軸より**下**にある部分の $x$ である。
> - $y = 0$ となる $x$ は、放物線と $x$ 軸の**交点**の $x$ である。

放物線が $x$ 軸と交わる点の $x$ 座標は、$y = 0$、すなわち $ax^2 + bx + c = 0$ の解である。二次方程式 $ax^2 + bx + c = 0$ の実数解の個数は、前提の章[判別式](../../algebra/discriminant/)で学んだとおり、判別式 $D = b^2 - 4ac$ の符号で決まる。よって二次不等式の解の形も、判別式 $D$ の符号で 3 通りに分かれる。

| 判別式 $D$ | $x$ 軸との交点 | 放物線の様子            |
| ---------- | -------------- | ----------------------- |
| $D > 0$    | 異なる 2 点    | $x$ 軸を 2 か所で横切る |
| $D = 0$    | 1 点で接する   | $x$ 軸に接する（頂点）  |
| $D < 0$    | 交点なし       | $x$ 軸の上に浮く        |

以下、$a > 0$（下に凸）のもとで、3 つの場合を順に調べる。

## $D > 0$ の場合

### 解の形

判別式 $D > 0$ のとき、$ax^2 + bx + c = 0$ は異なる 2 つの実数解を持つ。2 解を $\alpha$・$\beta$ とし、$\alpha < \beta$ とする。放物線 $y = ax^2 + bx + c$ は、$x$ 軸を $x = \alpha$ と $x = \beta$ の 2 点で横切る。$a > 0$ なので放物線は下に凸であり、2 交点の間で $x$ 軸より下、外側で上にある。

放物線と符号の対応を、$a > 0$・$D > 0$ の場合で表にまとめる。

| $x$ の範囲           | 放物線の上下   | $y$ の符号 |
| -------------------- | -------------- | ---------- |
| $x < \alpha$         | $x$ 軸より上   | $y > 0$    |
| $x = \alpha$         | $x$ 軸上の交点 | $y = 0$    |
| $\alpha < x < \beta$ | $x$ 軸より下   | $y < 0$    |
| $x = \beta$          | $x$ 軸上の交点 | $y = 0$    |
| $x > \beta$          | $x$ 軸より上   | $y > 0$    |

図で確かめる。例として $y = x^2 - 3x + 2$ を考える。$x^2 - 3x + 2 = (x - 1)(x - 2)$ なので、$x$ 軸との交点は $x = 1$ と $x = 2$ である。

### $> 0$ の解と $< 0$ の解

放物線の上下から、二次不等式の解が直ちに読める。$a > 0$・$D > 0$、2 解を $\alpha < \beta$ とする。

> $a > 0$・$D > 0$（2 解 $\alpha < \beta$）の二次不等式の解
>
> $$
> ax^2 + bx + c > 0 \iff x < \alpha \ \text{または}\ x > \beta
> $$
>
> $$
> ax^2 + bx + c < 0 \iff \alpha < x < \beta
> $$

記号 $\iff$ は、前提の章[一次不等式](../../algebra/linear-inequalities/)で導入した「同値である」と読む論理記号である。左右の主張が互いの言い換え、すなわち同じ条件であることを表す[^iff]。

例 $x^2 - 3x + 2 > 0$ は、放物線が $x$ 軸より上の範囲を問う。解は $x < 1$ または $x > 2$ である。一方 $x^2 - 3x + 2 < 0$ は、放物線が $x$ 軸より下の範囲を問う。解は $1 < x < 2$ である。2 つの解を数直線で示す。

### 等号を含む場合

不等号が $\ge$ や $\le$ のときは、交点 $x = \alpha$・$x = \beta$ で $y = 0$ となる点も解に含める。数直線では端点を白丸から黒丸へ変える。

$$
ax^2 + bx + c \ge 0 \iff x \le \alpha \ \text{または}\ x \ge \beta
$$

$$
ax^2 + bx + c \le 0 \iff \alpha \le x \le \beta
$$

## $D = 0$ の場合

### 接する放物線

判別式 $D = 0$ のとき、$ax^2 + bx + c = 0$ は重解を持つ。重解を $\alpha$ とすると、前提の章[判別式](../../algebra/discriminant/)より $\alpha = -\dfrac{b}{2a}$ であり、二次式は完全平方になる。

$$
ax^2 + bx + c = a(x - \alpha)^2
$$

放物線 $y = a(x - \alpha)^2$ は、頂点 $(\alpha, 0)$ で $x$ 軸に接する。$a > 0$ なら、$x = \alpha$ 以外のすべての $x$ で $y > 0$ となる。理由は、$(x - \alpha)^2$ が $x \neq \alpha$ で正、$x = \alpha$ でのみ $0$ になるためである。記号 $\neq$ は「等しくない」と読み、両辺が等しくないことを表す。

図で確かめる。例として $y = x^2 - 4x + 4 = (x - 2)^2$ を考える。頂点 $(2, 0)$ で $x$ 軸に接する。

### 各不等式の解

$a > 0$・$D = 0$（重解 $\alpha$）の場合、$y \ge 0$ がつねに成り立つ点に注意して解を読む。

> $a > 0$・$D = 0$（重解 $\alpha$）の二次不等式の解
>
> - $ax^2 + bx + c > 0$ の解は $x \neq \alpha$（$\alpha$ を除くすべての実数）。
> - $ax^2 + bx + c \ge 0$ の解はすべての実数。
> - $ax^2 + bx + c < 0$ の解はなし。
> - $ax^2 + bx + c \le 0$ の解は $x = \alpha$ の 1 点のみ。

例 $(x - 2)^2 > 0$ は、$x = 2$ で $0$、それ以外で正なので、解は $x \neq 2$ である。$(x - 2)^2 < 0$ は、$y$ が負になる $x$ が存在しないので、解はない。$(x - 2)^2 \le 0$ は、$y = 0$ となる $x = 2$ だけが解である。

## $D < 0$ の場合

### x 軸の上に浮く放物線

判別式 $D < 0$ のとき、$ax^2 + bx + c = 0$ は実数解を持たない。よって放物線 $y = ax^2 + bx + c$ は $x$ 軸と交わらない。$a > 0$ なら、放物線は $x$ 軸の上に浮き、すべての $x$ で $y > 0$ となる。

理由を平方完成で確かめる。前提の章[判別式](../../algebra/discriminant/)で導いた式を使う。

$$
ax^2 + bx + c = a\left(x + \frac{b}{2a}\right)^2 - \frac{D}{4a}
$$

$a > 0$ かつ $D < 0$ のとき、$-\dfrac{D}{4a}$ は正である。第 1 項 $a\left(x + \dfrac{b}{2a}\right)^2$ は $0$ 以上である。よって右辺はつねに正の値以上となり、$y > 0$ がすべての $x$ で成り立つ。

図で確かめる。例として $y = x^2 + 2 = x^2 - 0 \cdot x + 2$ を考える。$D = 0^2 - 4 \cdot 1 \cdot 2 = -8 < 0$ なので、$x$ 軸と交わらない。

### 各不等式の解

$a > 0$・$D < 0$ の場合、$y > 0$ がすべての $x$ で成り立つ。

> $a > 0$・$D < 0$ の二次不等式の解
>
> - $ax^2 + bx + c > 0$ の解はすべての実数。
> - $ax^2 + bx + c \ge 0$ の解はすべての実数。
> - $ax^2 + bx + c < 0$ の解はなし。
> - $ax^2 + bx + c \le 0$ の解はなし。

例 $x^2 + 2 > 0$ は、すべての $x$ で成り立つので解はすべての実数である。$x^2 + 2 < 0$ は、$y$ が負になる $x$ が存在しないので解はない。

### 解き方のまとめ

$a > 0$ の二次不等式を解く手順をまとめる。

1. 必要なら両辺に $-1$ を掛け、$x^2$ の係数を正にそろえる（向きの反転に注意）。
2. 判別式 $D = b^2 - 4ac$ の符号を調べる。
3. $D > 0$ なら 2 解 $\alpha < \beta$ を求め、放物線の上下から解を読む。
4. $D = 0$ なら重解 $\alpha$ を求め、$y \ge 0$ を使って解を読む。
5. $D < 0$ なら $y > 0$ がつねに成り立つことから解を読む。

## 絶対値を含む不等式

### 絶対値と距離の復習

前提の章を通じて、絶対値 $\lvert x \rvert$ は数直線上で点 $x$ と原点 $0$ との距離を表すことを学んだ。距離としての意味を使うと、絶対値の不等式を数直線上の範囲の問題に読み替えられる。本節では $a > 0$ とする。

基本となる 2 つの不等式は、距離の解釈から次のように解ける。

$$
\lvert x \rvert < a \iff -a < x < a
$$

$$
\lvert x \rvert > a \iff x < -a \ \text{または}\ x > a
$$

$\lvert x \rvert < a$ は「原点からの距離が $a$ 未満」を表し、$-a$ と $a$ の間の 1 つの範囲になる。$\lvert x \rvert > a$ は「原点からの距離が $a$ を超える」を表し、$-a$ より左と $a$ より右の離れた 2 つの範囲になる。$a = 2$ の場合を数直線で示し、2 つの解の形を見比べる。

### 一般の式の絶対値

絶対値の中身が $x$ そのものでなく、$x$ の式 $f(x)$ になる不等式も、同じ規則で解ける。中身 $f(x)$ をひとかたまりとみなし、規則を当てはめる。$a > 0$ とする。

> 絶対値を含む不等式の同値変形（$a > 0$）
>
> $$
> \lvert f(x) \rvert < a \iff -a < f(x) < a
> $$
>
> $$
> \lvert f(x) \rvert > a \iff f(x) < -a \ \text{または}\ f(x) > a
> $$

規則が成り立つ理由を述べる。$\lvert f(x) \rvert$ は、数直線上で点 $f(x)$ と原点との距離である。$\lvert f(x) \rvert < a$ は「点 $f(x)$ と原点との距離が $a$ 未満」を表す。距離が $a$ 未満の点は $-a$ と $a$ の間にあるので、$-a < f(x) < a$ となる。$\lvert f(x) \rvert > a$ も同様に距離の解釈から得られる。

変形の後は、$f(x)$ についての不等式を解く。$f(x)$ が 1 次式なら、前提の章[一次関数](../linear-functions/)で扱った 1 次式の計算で解ける。例として $\lvert 2x - 1 \rvert < 5$ を解く。

**変形.** $\lvert f(x) \rvert < a$ の規則を $f(x) = 2x - 1$・$a = 5$ に当てはめる。

$$
-5 < 2x - 1 < 5
$$

**各辺に同じ操作.** 連続した不等式 $-5 < 2x - 1 < 5$ は、$-5 < 2x - 1$ かつ $2x - 1 < 5$ の略記である。各不等式は、前提の章[一次不等式](../../algebra/linear-inequalities/)の基本性質で変形できる。両方に同じ操作をすると、まとめて 3 辺に同じ操作をした形になる[^chain]。各辺に $1$ を足す。

$$
-4 < 2x < 6
$$

各辺を正の数 $2$ で割る。$2 > 0$ なので向きは変わらない。

$$
-2 < x < 3
$$

解は $-2 < x < 3$ である。数直線で示す。

## 例題

### 例題 1

二次不等式 $x^2 - x - 6 > 0$ を解け。

**解法.** $x^2 - x - 6 = (x + 2)(x - 3)$ と因数分解する。$x^2 - x - 6 = 0$ の解は $x = -2$ と $x = 3$ である。係数 $a = 1 > 0$ なので、放物線は下に凸で、2 交点 $-2$・$3$ の外側で $x$ 軸より上にある。$> 0$ は $x$ 軸より上の範囲を問う。

$$
x < -2 \ \text{または}\ x > 3
$$

解は $x < -2$ または $x > 3$ である。

### 例題 2

二次不等式 $x^2 - x - 6 \le 0$ を解け。

**解法.** 例題 1 と同じく、$x^2 - x - 6 = (x + 2)(x - 3)$ で、交点は $x = -2$・$x = 3$ である。$\le 0$ は $x$ 軸より下、および交点を問う。下に凸の放物線は、2 交点の間で $x$ 軸より下にある。等号を含むので端点も解に入れる。

$$
-2 \le x \le 3
$$

解は $-2 \le x \le 3$ である。

### 例題 3

二次不等式 $x^2 + 4x + 4 > 0$ を解け。

**解法.** $x^2 + 4x + 4 = (x + 2)^2$ と完全平方になる。判別式は $D = 4^2 - 4 \cdot 1 \cdot 4 = 0$ で、放物線は頂点 $(-2, 0)$ で $x$ 軸に接する。$a = 1 > 0$ なので、$x = -2$ で $y = 0$、それ以外で $y > 0$ である。$> 0$ は等号を含まないので、$y = 0$ となる $x = -2$ は解に入らない。

$$
x \neq -2
$$

解は $x = -2$ を除くすべての実数である。

### 例題 4

二次不等式 $x^2 + x + 1 > 0$ を解け。

**解法.** 判別式を調べる。$D = 1^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 < 0$ である。$a = 1 > 0$ かつ $D < 0$ なので、放物線は $x$ 軸の上に浮き、すべての $x$ で $y > 0$ となる。

$$
\text{解はすべての実数}
$$

すべての実数が解である。

### 例題 5

$x^2$ の係数が負の二次不等式 $-x^2 + 2x + 3 \ge 0$ を解け。

**解法.** $x^2$ の係数を正にそろえる。両辺に $-1$ を掛けると、不等号の向きが反転する。

$$
x^2 - 2x - 3 \le 0
$$

$x^2 - 2x - 3 = (x + 1)(x - 3)$ と因数分解する。交点は $x = -1$・$x = 3$ である。$\le 0$ は、下に凸の放物線が $x$ 軸より下、および交点を問う。

$$
-1 \le x \le 3
$$

解は $-1 \le x \le 3$ である。

### 例題 6

絶対値を含む不等式 $\lvert x + 1 \rvert \ge 2$ を解け。

**解法.** $\lvert f(x) \rvert \ge a$ は、$\lvert f(x) \rvert > a$ の規則に等号を加えた形である。$f(x) = x + 1$・$a = 2$ に当てはめる。

$$
x + 1 \le -2 \ \text{または}\ x + 1 \ge 2
$$

各不等式から $1$ を引く。

$$
x \le -3 \ \text{または}\ x \ge 1
$$

解は $x \le -3$ または $x \ge 1$ である。離れた 2 つの範囲になり、端点は $\le$・$\ge$ なので含む。数直線では端点を黒丸で表す。

## まとめ

本章は、二次不等式と絶対値を含む不等式を、前提の章だけから自己完結で扱った。要点を振り返る。

- **二次不等式**は、二次関数 $y = ax^2 + bx + c$ の値の符号を問う不等式である。$y > 0$ は放物線が $x$ 軸より上、$y < 0$ は下の範囲に当たる。
- $x^2$ の係数は、両辺に $-1$ を掛けて正にそろえる。負の数を掛けると不等号の向きは反転する。
- 解の形は、判別式 $D = b^2 - 4ac$ の符号で 3 通りに分かれる。
  - $D > 0$：2 交点 $\alpha < \beta$。$> 0$ の解は $x < \alpha$ または $x > \beta$、$< 0$ の解は $\alpha < x < \beta$ である。
  - $D = 0$：頂点で接する。$> 0$ の解は $x \neq \alpha$、$< 0$ の解はなしである。
  - $D < 0$：交点なし。$> 0$ の解はすべての実数、$< 0$ の解はなしである。
- 絶対値を含む不等式は、距離としての意味から解ける。$a > 0$ のとき、$\lvert f(x) \rvert < a \iff -a < f(x) < a$、$\lvert f(x) \rvert > a \iff f(x) < -a$ または $f(x) > a$ である。

不等式は、関数の値が取る範囲や、量に課す条件の記述に欠かせない。本章で初等関数のうち多項式の一部を扱い終えた。次の章[多項式関数と有理関数](../polynomial-and-rational-functions/)では、3 次以上の多項式関数と、分数で表す有理関数へ進む。

[^sign-flip]: 不等式の両辺に同じ負の数を掛ける、または同じ負の数で割ると、不等号の向きが反転する。規則は、前提の手前の代数カテゴリで扱う一次不等式の基本性質に基づく。$a < b$ のとき $b - a > 0$ であり、負の数 $c$ を掛けると $c(b - a) < 0$、すなわち $ac > bc$ となる。

[^iff]: 記号 $\iff$ は、左の主張から右の主張が導け、かつ右の主張から左の主張も導けることを表す。同値変形では、変形の前後で解の集合が一切変わらない。二次不等式や絶対値の不等式の解法は、解の集合を変えない同値変形の連鎖である。

[^chain]: 連続した不等式 $p < q < r$ は、$p < q$ かつ $q < r$ を 1 つにまとめた略記である。3 辺すべてに同じ数を足す操作は、$p < q$ と $q < r$ の両方に同じ数を足す操作に等しい。各辺に同じ正の数を掛ける、または同じ正の数で割る操作も同様である。よって、各不等式が向きを保つ操作なら、連続した不等式も向きを保つ。
