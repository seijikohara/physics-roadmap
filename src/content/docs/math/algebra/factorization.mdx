## 前提

本章の前提は、前の章[多項式の展開](../polynomial-expansion/)である。多項式・項・係数・定数項・次数・同類項の定義と、分配法則 $a(b + c) = ab + ac$、および乗法公式を使う。

[多項式の展開](../polynomial-expansion/)では、積を和の形に書き直す**展開**を学んだ。本章は逆向きの操作を扱う。和を積の形に書き直す操作を**因数分解**と呼ぶ。

## 学習目標

本章を読むと、次の概念と操作を使えるようになる。

- 因数分解の定義と、展開との逆向きの関係
- 共通因数のくくり出しによる因数分解
- 乗法公式を逆向きに読んだ、公式による因数分解
- 立方の和 $a^3 + b^3$ と立方の差 $a^3 - b^3$ の因数分解
- たすき掛けによる $acx^2 + (ad + bc)x + bd$ の因数分解
- 共通因数のくくり出しと公式の組み合わせによる因数分解
- 置換による因数分解

## 因数分解とは何か

**因数分解**とは、1 つの多項式を、2 つ以上の多項式の積の形に書き直す操作である。積を作る各式を**因数**と呼ぶ。因数の定義は前の章[多項式の展開](../polynomial-expansion/)と同じである。本章では、因数の係数を整数の範囲に限る[^coefficient-range]。

展開と因数分解は、互いに逆向きの操作である。次の 1 組の式で両者を比べる。

- **展開**は、積 $(x + 2)(x + 3)$ を和 $x^2 + 5x + 6$ に書き直す。
- **因数分解**は、和 $x^2 + 5x + 6$ を積 $(x + 2)(x + 3)$ に書き直す。

因数分解の結果が正しいかどうかは、得られた積を展開して確かめられる。展開した結果がもとの多項式に戻れば、因数分解は正しい。展開による検算は、どの因数分解にも使える。本章では、たすき掛けのように誤りやすい場合を中心に、展開で確かめる手順を示す。読者は他の例題や演習でも、同じ方法で結果を検算できる。

## 共通因数のくくり出し

多項式の各項に共通して現れる因数を**共通因数**と呼ぶ。共通因数は、分配法則を逆向きに使ってくくり出せる。

分配法則 $a(b + c) = ab + ac$ を、右辺から左辺へ読む。

$$
ab + ac = a(b + c)
$$

右辺 $ab + ac$ の 2 つの項は、どちらも因数 $a$ を含む。共通因数 $a$ をくくり出すと、積 $a(b + c)$ になる。共通因数のくくり出しは、最も基本的な因数分解である。

### 共通因数の見つけ方

**最大公約数**とは、2 つ以上の整数に共通する約数のうち最も大きいものである。例えば $6$ と $9$ の公約数は $1$ と $3$ であり、最大公約数は $3$ である。

共通因数のくくり出しは、次の手順で進める。

1. 各項を因数の積に分け、すべての項に共通する因数を探す。
2. 係数の共通因数は、各項の係数の最大公約数を取る。
3. 文字の共通因数は、各項に共通して現れる文字を、最も低い次数まで取る。
4. 共通因数でくくり、残りを和の形でかっこに入れる。

### くくり出しの例

$6x^2 + 9x$ を因数分解する。

- 係数 $6$ と $9$ の最大公約数は $3$ である。
- 文字 $x$ は、$6x^2 = 6 \cdot x \cdot x$ と $9x = 9 \cdot x$ の両方に現れる。共通する次数は $1$ である。
- 共通因数は $3x$ である。

共通因数 $3x$ でくくり出す。

$$
6x^2 + 9x = 3x \cdot 2x + 3x \cdot 3 = 3x(2x + 3)
$$

結果を展開して確かめる。$3x(2x + 3) = 6x^2 + 9x$ となり、もとの式に戻る。

## 公式による因数分解

前の章[多項式の展開](../polynomial-expansion/)の乗法公式は、左辺と右辺を入れ替えると、そのまま因数分解の公式になる。展開公式が「積 $\to$ 和」を与えるのに対し、因数分解の公式は「和 $\to$ 積」を与える。

次の表に、本章で扱う因数分解の公式をまとめる。文字 $a$・$b$・$x$ は、どんな数や式でもよいことを表す。文字の役割は公式ごとに独立である。各公式は単独で読み、文字の対応をその都度定める。

| 番号 | 因数分解の公式                         | 導出の根拠                   |
| ---- | -------------------------------------- | ---------------------------- |
| (1)  | $a^2 + 2ab + b^2 = (a + b)^2$          | 前章の展開公式を逆向きに読む |
| (2)  | $a^2 - 2ab + b^2 = (a - b)^2$          | 前章の展開公式を逆向きに読む |
| (3)  | $a^2 - b^2 = (a + b)(a - b)$           | 前章の展開公式を逆向きに読む |
| (4)  | $x^2 + (a + b)x + ab = (x + a)(x + b)$ | 前章の展開公式を逆向きに読む |
| (5)  | $a^3 + b^3 = (a + b)(a^2 - ab + b^2)$  | 本章で展開から確認する       |
| (6)  | $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$  | 本章で展開から確認する       |

以下、各公式の使い方を示す。公式 (1) から (4) は、前の章の展開公式を逆向きに読んだものである。公式 (5) と (6) は、本章で改めて展開から確かめる。

### 公式 (1) と (2)：平方の形

公式 (1) と (2) は、$a^2 + 2ab + b^2$ や $a^2 - 2ab + b^2$ の形を、平方 $(a + b)^2$ や $(a - b)^2$ に書き直す。

平方の形を見分ける目印は、次の 3 点である。

- 両端の 2 項が、ある式の 2 乗の形になっている。例えば $a^2$ と $b^2$ である。
- 中央の項が、両端の式の積の 2 倍になっている。例えば $2ab$ である。
- 中央の項の符号が正なら $(a + b)^2$、負なら $(a - b)^2$ になる。

例として $x^2 + 6x + 9$ を因数分解する。

- 両端の項は $x^2$ と $9 = 3^2$ である。$a = x$、$b = 3$ と見る。
- 中央の項 $6x$ は、$2 \cdot x \cdot 3 = 6x$ に等しい。平方の形に当てはまる。
- 中央の符号は正なので、$(a + b)^2$ の形になる。

$$
x^2 + 6x + 9 = (x + 3)^2
$$

結果を展開すると $(x + 3)^2 = x^2 + 6x + 9$ となり、もとの式に戻る。

### 公式 (3)：2 乗の差

公式 (3) は、2 乗の差 $a^2 - b^2$ を、和と差の積 $(a + b)(a - b)$ に書き直す。

例として $x^2 - 25$ を因数分解する。$25 = 5^2$ なので、$a = x$、$b = 5$ と見る。

$$
x^2 - 25 = x^2 - 5^2 = (x + 5)(x - 5)
$$

結果を展開すると $(x + 5)(x - 5) = x^2 - 25$ となり、もとの式に戻る。

### 公式 (4)：1 次の係数が 1 の 2 次式

公式 (4) は、$x^2 + (a + b)x + ab$ の形を、$(x + a)(x + b)$ に書き直す。$x^2$ の係数が $1$ の 2 次式に使う。

公式 (4) を使う鍵は、次の 2 条件を満たす 2 数 $a$・$b$ を見つけることである。

- 2 数の積 $ab$ が、定数項に等しい。
- 2 数の和 $a + b$ が、$x$ の係数に等しい。

例として $x^2 + 5x + 6$ を因数分解する。積が $6$、和が $5$ になる 2 数を探す。積が $6$ になる整数の組を表に並べ、和を調べる。

| 2 数の組     | 積  | 和   |
| ------------ | --- | ---- |
| $1$ と $6$   | 6   | 7    |
| $2$ と $3$   | 6   | 5    |
| $-1$ と $-6$ | 6   | $-7$ |
| $-2$ と $-3$ | 6   | $-5$ |

積が $6$、和が $5$ になる組は $2$ と $3$ である。$a = 2$、$b = 3$ と定める。

$$
x^2 + 5x + 6 = (x + 2)(x + 3)
$$

結果を展開すると $(x + 2)(x + 3) = x^2 + 5x + 6$ となり、もとの式に戻る。

定数項が負の場合は、2 数の符号が異なる。例として $x^2 - x - 12$ を因数分解する。積が $-12$、和が $-1$ になる 2 数を探す。積が負なので、2 数の符号は異なる。組は $3$ と $-4$ である。

$$
x^2 - x - 12 = (x + 3)(x - 4)
$$

### 公式 (5) と (6)：立方の和と差

公式 (5) と (6) は、立方の和 $a^3 + b^3$ と立方の差 $a^3 - b^3$ を因数分解する。**立方**とは 3 乗を指す。前の章では立方の和と差の展開を扱わなかったため、本章で展開から確かめる。

公式 (5) の右辺 $(a + b)(a^2 - ab + b^2)$ を展開する。すべての組み合わせで掛け合わせる。

$$
(a + b)(a^2 - ab + b^2) = a^3 - a^2 b + ab^2 + a^2 b - ab^2 + b^3
$$

同類項を整理する。$-a^2 b$ と $+a^2 b$ が打ち消し合い、$+ab^2$ と $-ab^2$ も打ち消し合う。

$$
(a + b)(a^2 - ab + b^2) = a^3 + b^3
$$

右辺を展開した結果が $a^3 + b^3$ に等しい。よって公式 (5) が成り立つ。公式 (6) も同様に確かめられる。

$$
(a - b)(a^2 + ab + b^2) = a^3 + a^2 b + ab^2 - a^2 b - ab^2 - b^3 = a^3 - b^3
$$

例として、立方の和 $x^3 + 8$ を因数分解する。$8 = 2^3$ なので、$a = x$、$b = 2$ と見る。公式 (5) に代入する。

$$
x^3 + 8 = x^3 + 2^3 = (x + 2)(x^2 - 2x + 4)
$$

立方の差 $x^3 - 1$ も同様に因数分解する。$1 = 1^3$ なので、$a = x$、$b = 1$ と見る。公式 (6) に代入する[^second-factor]。

$$
x^3 - 1 = x^3 - 1^3 = (x - 1)(x^2 + x + 1)
$$

## たすき掛けによる因数分解

$x^2$ の係数が $1$ でない 2 次式 $acx^2 + (ad + bc)x + bd$ は、公式 (4) では因数分解できない。次の形の積に分解する。

$$
acx^2 + (ad + bc)x + bd = (ax + b)(cx + d)
$$

右辺は、前の章の展開公式 $(ax + b)(cx + d) = acx^2 + (ad + bc)x + bd$ の左辺と右辺を入れ替えたものである。

係数 $a$・$b$・$c$・$d$ を見つける作業を、**たすき掛け**と呼ぶ。たすき掛けは、次の関係を満たす 4 数を探す手順である。

- $a$ と $c$ の積 $ac$ が、$x^2$ の係数に等しい。
- $b$ と $d$ の積 $bd$ が、定数項に等しい。
- $a \cdot d$ と $b \cdot c$ の和 $ad + bc$ が、$x$ の係数に等しい。

### たすき掛けの図

たすき掛けは、係数を 2 行 2 列に並べた図で進める。左の列へ $a$・$c$ を縦に並べ、右の列へ $b$・$d$ を縦に並べる。下の表は、4 数の配置と斜めの積を表す。

| 行  | 左の列 | 右の列 | 斜めの積     |
| --- | ------ | ------ | ------------ |
| 上  | $a$    | $b$    | $a \times d$ |
| 下  | $c$    | $d$    | $c \times b$ |

「たすき掛け」の名は、左上の $a$ から右下の $d$ へ、左下の $c$ から右上の $b$ へと斜めに線を引くと、たすきのような交差ができることに由来する。配置から 3 種類の積を読み取る。

- **縦の積（左）**: $a \times c = ac$。$x^2$ の係数に等しい。
- **縦の積（右）**: $b \times d = bd$。定数項に等しい。
- **斜めの積の和**: $a \times d$ と $c \times b$ の和 $ad + bc$。$x$ の係数に等しい。

3 種類の積が条件を満たす 4 数を見つけると、因数分解 $(ax + b)(cx + d)$ が定まる。

### たすき掛けの例

$2x^2 + 7x + 3$ を因数分解する。$x^2$ の係数と定数項を因数の積に分け、組を試す。係数や定数項が小さい数や素数のときは、分け方が少ないため、最も簡単な組から試すと速い。

- $x^2$ の係数は $2$ である。積 $ac = 2$ となる組は $a = 1$、$c = 2$ だけである。
- 定数項は $3$ である。積 $bd = 3$ となる組は $b = 1$、$d = 3$ か $b = 3$、$d = 1$ である。
- 斜めの和 $ad + bc$ が $x$ の係数 $7$ に等しくなる組を探す。

候補を表で調べる。

| $a$ | $c$ | $b$ | $d$ | $ad + bc$                   | $x$ の係数 $7$ と一致するか |
| --- | --- | --- | --- | --------------------------- | --------------------------- |
| 1   | 2   | 1   | 3   | $1 \cdot 3 + 2 \cdot 1 = 5$ | 一致しない                  |
| 1   | 2   | 3   | 1   | $1 \cdot 1 + 2 \cdot 3 = 7$ | 一致する                    |

$a = 1$、$c = 2$、$b = 3$、$d = 1$ が条件を満たす。確定した 4 数を、たすき掛けの配置に当てはめる。

| 行  | 左の列  | 右の列  | 斜めの積              |
| --- | ------- | ------- | --------------------- |
| 上  | $a = 1$ | $b = 3$ | $ad = 1 \times 1 = 1$ |
| 下  | $c = 2$ | $d = 1$ | $bc = 3 \times 2 = 6$ |

3 種類の積を確かめる。縦の積は $ac = 1 \times 2 = 2$ と $bd = 3 \times 1 = 3$ である。斜めの和は $ad + bc = 1 + 6 = 7$ である。3 つとも条件に合う。確定した 4 数を $(ax + b)(cx + d)$ に代入する。

$$
2x^2 + 7x + 3 = (x + 3)(2x + 1)
$$

結果を展開して確かめる。

$$
(x + 3)(2x + 1) = 2x^2 + x + 6x + 3 = 2x^2 + 7x + 3
$$

もとの式に戻る。因数分解は正しい。

## 複数の手法の組み合わせ

1 つの多項式に、複数の手法を組み合わせて使う場合がある。手順の基本は、共通因数のくくり出しを最初に試すことである。

### 共通因数をくくってから公式を使う

共通因数をくくり出した後、残った式にさらに公式を使える場合がある。例として $2x^2 - 8$ を因数分解する。

まず共通因数 $2$ をくくり出す。

$$
2x^2 - 8 = 2(x^2 - 4)
$$

かっこの中 $x^2 - 4$ は、2 乗の差である。$4 = 2^2$ なので、公式 (3) を使える。

$$
x^2 - 4 = (x + 2)(x - 2)
$$

2 つの結果を合わせる。

$$
2x^2 - 8 = 2(x + 2)(x - 2)
$$

共通因数のくくり出しを先に行うと、残った式が公式の形になり、因数分解を最後まで進められる。

### 置換による因数分解

式の一部のまとまりを 1 つの文字に置き換えると、見慣れた形になる場合がある。まとまりを別の文字に置き換える操作を**置換**と呼ぶ。

例として $(x + y)^2 + 3(x + y) + 2$ を因数分解する。まとまり $x + y$ を文字 $A$ に置き換える。

$$
A = x + y
$$

置き換えると、式は $A$ の 2 次式になる。

$$
(x + y)^2 + 3(x + y) + 2 = A^2 + 3A + 2
$$

$A^2 + 3A + 2$ は、公式 (4) で因数分解できる。積が $2$、和が $3$ になる 2 数は $1$ と $2$ である。

$$
A^2 + 3A + 2 = (A + 1)(A + 2)
$$

最後に、$A$ をもとの $x + y$ に戻す。

$$
(A + 1)(A + 2) = (x + y + 1)(x + y + 2)
$$

置換は、複雑な式を見慣れた形に直し、公式を使えるようにする手法である。因数分解の後は、置いた文字をもとの式に戻す。

## 例題

### 例題 1

$4x^2 + 6x$ を因数分解せよ。

**解法.** 共通因数をくくり出す。係数 $4$ と $6$ の最大公約数は $2$ である。文字 $x$ は両方の項に現れ、共通する次数は $1$ である。共通因数は $2x$ である。

$$
4x^2 + 6x = 2x \cdot 2x + 2x \cdot 3 = 2x(2x + 3)
$$

### 例題 2

$x^2 - 49$ を因数分解せよ。

**解法.** 形は 2 乗の差である。$49 = 7^2$ なので、公式 (3) $a^2 - b^2 = (a + b)(a - b)$ に当たる。$a = x$、$b = 7$ と対応させる。

$$
x^2 - 49 = x^2 - 7^2 = (x + 7)(x - 7)
$$

### 例題 3

$x^2 + 7x + 12$ を因数分解せよ。

**解法.** $x^2$ の係数が $1$ なので、公式 (4) $x^2 + (a + b)x + ab = (x + a)(x + b)$ に当たる。積が $12$、和が $7$ になる 2 数を探す。組は $3$ と $4$ である。

$$
x^2 + 7x + 12 = (x + 3)(x + 4)
$$

### 例題 4

$3x^2 + 10x + 8$ を因数分解せよ。

**解法.** $x^2$ の係数が $1$ でないため、たすき掛けを使う。$ac = 3$、$bd = 8$、$ad + bc = 10$ を満たす 4 数を探す。$a = 1$、$c = 3$ とし、$b$・$d$ の組を調べる。

| $a$ | $c$ | $b$ | $d$ | $ad + bc$                    |
| --- | --- | --- | --- | ---------------------------- |
| 1   | 3   | 2   | 4   | $1 \cdot 4 + 3 \cdot 2 = 10$ |

$a = 1$、$c = 3$、$b = 2$、$d = 4$ が条件を満たす。

$$
3x^2 + 10x + 8 = (x + 2)(3x + 4)
$$

展開して確かめると $(x + 2)(3x + 4) = 3x^2 + 4x + 6x + 8 = 3x^2 + 10x + 8$ となる。

### 例題 5

$3x^2 - 12$ を因数分解せよ。

**解法.** まず共通因数 $3$ をくくり出す。

$$
3x^2 - 12 = 3(x^2 - 4)
$$

かっこの中 $x^2 - 4$ は 2 乗の差である。公式 (3) で因数分解する。

$$
x^2 - 4 = (x + 2)(x - 2)
$$

合わせて次の結果になる。

$$
3x^2 - 12 = 3(x + 2)(x - 2)
$$

両端の項は $x^2$ と $25 = 5^2$ である。中央の項 $10x$ は $2 \cdot x \cdot 5$ に等しい。平方の形に当てはまる。中央の符号は正なので、公式 (1) $a^2 + 2ab + b^2 = (a + b)^2$ に当たる。$a = x$、$b = 5$ と対応させる。

$$
x^2 + 10x + 25 = (x + 5)^2
$$

</details>

### 問題 3

$x^2 - 81$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

形は 2 乗の差である。$81 = 9^2$ なので、公式 (3) $a^2 - b^2 = (a + b)(a - b)$ に当たる。$a = x$、$b = 9$ と対応させる。

$$
x^2 - 81 = x^2 - 9^2 = (x + 9)(x - 9)
$$

</details>

### 問題 4

$x^2 - 2x - 15$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

$x^2$ の係数が $1$ なので、公式 (4) $x^2 + (a + b)x + ab = (x + a)(x + b)$ に当たる。積が $-15$、和が $-2$ になる 2 数を探す。積が負なので、2 数の符号は異なる。組は $3$ と $-5$ である。

$$
x^2 - 2x - 15 = (x + 3)(x - 5)
$$

</details>

### 問題 5

$2x^2 + 5x + 2$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

$x^2$ の係数が $1$ でないため、たすき掛けを使う。$ac = 2$、$bd = 2$、$ad + bc = 5$ を満たす 4 数を探す。

| $a$ | $c$ | $b$ | $d$ | $ad + bc$                   |
| --- | --- | --- | --- | --------------------------- |
| 1   | 2   | 2   | 1   | $1 \cdot 1 + 2 \cdot 2 = 5$ |

$a = 1$、$c = 2$、$b = 2$、$d = 1$ が条件を満たす。

$$
2x^2 + 5x + 2 = (x + 2)(2x + 1)
$$

展開して確かめると $(x + 2)(2x + 1) = 2x^2 + x + 4x + 2 = 2x^2 + 5x + 2$ となる。

</details>

### 問題 6

$5x^2 - 20$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

まず共通因数 $5$ をくくり出す。

$$
5x^2 - 20 = 5(x^2 - 4)
$$

かっこの中 $x^2 - 4$ は 2 乗の差である。公式 (3) で因数分解する。$4 = 2^2$ である。

$$
x^2 - 4 = (x + 2)(x - 2)
$$

合わせて次の結果になる。

$$
5x^2 - 20 = 5(x + 2)(x - 2)
$$

</details>

### 問題 7

$x^3 - 27$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

形は立方の差である。$27 = 3^3$ なので、公式 (6) $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$ に当たる。$a = x$、$b = 3$ と対応させる。

$$
x^3 - 27 = x^3 - 3^3 = (x - 3)(x^2 + 3x + 9)
$$

</details>

### 問題 8

$(x + 1)^2 - 4(x + 1) + 3$ を因数分解せよ。

<details>
<summary>解答を表示</summary>

まとまり $x + 1$ を文字 $A$ に置き換える。

$$
A = x + 1
$$

置き換えると、式は $A$ の 2 次式になる。

$$
(x + 1)^2 - 4(x + 1) + 3 = A^2 - 4A + 3
$$

公式 (4) で因数分解する。積が $3$、和が $-4$ になる 2 数は $-1$ と $-3$ である。

$$
A^2 - 4A + 3 = (A - 1)(A - 3)
$$

$A$ をもとの $x + 1$ に戻す。

$$
(A - 1)(A - 3) = (x + 1 - 1)(x + 1 - 3) = x(x - 2)
$$

</details>

## まとめ

本章は、多項式の因数分解を展開の逆操作として自己完結で扱った。要点を振り返る。

- 因数分解は、1 つの多項式を 2 つ以上の多項式の積に書き直す操作である。展開と逆向きの関係にある。
- 共通因数のくくり出しは、分配法則を逆向きに使う、最も基本的な因数分解である。
- 乗法公式の左辺と右辺を入れ替えると、因数分解の公式になる。
- $x^2$ の係数が $1$ でない 2 次式は、たすき掛けで因数分解する。
- 共通因数のくくり出しを最初に試し、残った式に公式や置換を組み合わせる。
- 因数分解の結果は、展開してもとの式に戻るかどうかで確かめられる。

[^coefficient-range]: 因数の係数として許す範囲を広げると、分解がさらに進む場合もある。例えば $x^2 - 2$ は、整数係数では分解できない。分数で表せない数である無理数を許せば $(x + \sqrt{2})(x - \sqrt{2})$ に分解できる。なお $\sqrt{2}$ は 2 乗すると $2$ になる正の数を表す。係数として何を許すかで、因数分解の結果は変わる。
